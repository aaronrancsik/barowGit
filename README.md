## barowGit

A module to display the current git branch in [barow](https://github.com/doums/barow).

### usage

```
" .vimrc/init.vim
" ...

Plug 'doums/barowGit'
" ...

let g:barow = {
      \  'modules': [
      \    [ 'barowGit#branch', 'StatusLine' ],
      \  ]
      \}
```
